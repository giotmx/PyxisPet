SISTEMA PARA ADOÇÃO DE ANIMAIS - PYXIS PET

O projeto tem como principal objetivo otimizar buscas e pesquisas dentro de uma organização validando informações de forma eficiente, reunindo os dados possíveis dos animais e de seus responsáveis.

O Pyxis Pet foi desenvolvido com o objetivo para ajudar as instituições cadastrando os animais que foram resgatados de maneira eficiente, simples e prática. Visando a diminuição de banco de dados físicos, além do mais o Pyxis Pet auxiliará na redução do uso de papéis.
